# 剧本杀售后展示网站

一个单页网站，用于展示剧本杀售后内容，支持自定义背景图和 YouTube 背景音乐。

## 使用方法

1. 双击打开 `index.html`，或在浏览器中打开该文件
2. 在右侧「编辑面板」中：
   - 输入**标题**和**售后主体文本**
   - 上传**背景图片**（支持拖拽）
   - 粘贴 **YouTube 音乐链接**
3. 点击「**应用预览**」查看效果
4. 点击「**进入体验**」开始播放背景音乐

## 功能说明

| 功能 | 说明 |
|------|------|
| 文本展示 | 支持多行售后说明、复盘、彩蛋等 |
| 背景图片 | 本地上传，即时预览 |
| 背景音乐 | 支持 `youtube.com` / `youtu.be` 链接，循环播放 |
| 本地保存 | 将配置保存到浏览器，下次可「读取本地」恢复 |
| 音乐控制 | 左下角可暂停/继续播放 |

## 关于自动播放

现代浏览器要求用户先进行一次点击操作，才允许带声音自动播放。因此页面会先显示「进入体验」按钮，点击后音乐才会开始播放。这是浏览器的安全策略，无法绕过。

## 支持的 YouTube 链接格式

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`

## 提示

- 背景图过大可能导致「保存到本地」失败，建议使用压缩后的图片（1MB 以内较稳妥）
- 收起编辑面板可全屏欣赏展示效果
