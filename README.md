# 剧本杀售后展示网站

一个单页网站，用于展示剧本杀售后内容，支持自定义背景图和 YouTube 背景音乐。

## 使用方法

1. 双击打开 `index.html`，或在浏览器中打开该文件
2. 在右侧「编辑面板」中：
   - 输入**标题**和**售后主体文本**
   - 上传**背景图片**（支持拖拽）
   - 粘贴 **YouTube 音乐链接**
3. 点击「**应用预览**」查看效果
4. 点击「**进入体验**」开始播放背景音乐

## 功能说明

| 功能 | 说明 |
|------|------|
| 文本展示 | 支持多行售后说明、复盘、彩蛋等 |
| 背景图片 | 本地上传，即时预览 |
| 背景音乐 | 支持 `youtube.com` / `youtu.be` 链接，循环播放 |
| 本地保存 | 将配置保存到浏览器，下次可「读取本地」恢复 |
| 音乐控制 | 左下角可暂停/继续播放 |

## 分享链接

1. 编辑好内容后，点击「**生成分享链接**」
2. 点击「**复制链接**」发给玩家
3. 对方打开链接后会**自动展示相同内容**，并**尝试自动播放**背景音乐

分享链接会把标题、文本、背景图、音乐链接编码在 URL 中。若本地上传的背景图过大，建议使用「**背景图网络链接**」（如图床 URL）再生成分享链接。

## 关于自动播放

现代浏览器可能仍要求用户点击一次页面才允许播放声音。若自动播放失败，页面底部会显示「点击任意处播放背景音乐」提示。

## 支持的 YouTube 链接格式

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`

## 提示

- 背景图过大可能导致「保存到本地」失败，建议使用压缩后的图片（1MB 以内较稳妥）
- 收起编辑面板可全屏欣赏展示效果
